# 🎮 Gamedesign Concepts

Личные заметки по геймдизайну — концепты игр, механики, циклы прогрессии.

---

## 📁 Концепты

### 🌱 Seedling Sanctuary
Фоновая игра про восстановление заброшенной теплицы. Растения растут в реальном времени, коллекционирование стадий роста, система улучшений и новых зон.

### 🎬 Starlight Drive-In
Симулятор автокинотеатра. Управляешь парковкой и показом, машины-зрители живут своей жизнью. Прокачка проектора, кафе, репутация — от заброшенной площадки до легендарного арт-хауса.

### 🌑 Морок
Survival horror на основе славянского фольклора. Выживание в отдаленной деревне, система ритуалов, бестиарий существ, моральные выборы и несколько концовок.
